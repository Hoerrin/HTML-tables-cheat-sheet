HTML tables cheat sheet

Tag Name Description
table 123 Defines a table
tr 123 Defines a row in a table
td 123 Defines a cell in a table
th 123 Defines a header cell in a table
caption 123 Defines a table caption
colgroup 123 Specifies a group of one or more columns in a table for formatting
col 123 Specifies column properties for each column within a element
thead 123 Groups the header content in a table
tbody 123 Groups the body content in a table
tfoot 123 Groups the footer content in a table
